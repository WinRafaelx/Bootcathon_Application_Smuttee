## 📦 Setup
We use 
- [Vite](https://vitejs.dev/) + React
- [mongodb](https://www.mongodb.com/) for our database
### Install dependencies

```sh
npm install
```

### 🔨 Development
For frontend run :
```sh
npm run dev
```
For backend run :
```sh
npm start
```
